**Emojo** is a `emoji search tool`, created with the sole purpose of practicing API calls and React Hooks and all of that new stuff.

## Installing emojo

To install **emojo**:

```
1. Clone the repository
```
```
2. yarn add or npm install
```
```
3. yarn start or npm start
```
